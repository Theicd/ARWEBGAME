# בדיקה על טלפון

כדי לבדוק את המשחק על טלפון:
1. ודא שהשרת המקומי פועל על המחשב שלך (http://localhost:8080).
2. חבר את הטלפון לאותה רשת Wi-Fi של המחשב.
3. גלה את כתובת ה-IP של המחשב שלך (למשל, 192.168.1.100).
4. פתח את הדפדפן בטלפון ונווט אל `http://[IP של המחשב]:8080` (למשל, http://192.168.1.100:8080).
5. אפשר גישה למצלמה ולגירוסקופ כאשר תתבקש.
6. כוון את המצלמה אל ה-Hiro marker שהדפסת כדי לראות את המטבעות הווירטואליים.

בהצלחה!
