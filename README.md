# משחק מציאות רבודה (AR)

## הוראות הפעלה
1. ודא שיש לך שרת מקומי מותקן (כמו `http-server` דרך npm).
2. הפעל את השרת בתיקיית הפרויקט עם הפקודה: `http-server ./ar_game -p 8080`.
3. פתח את הדפדפן בכתובת `http://localhost:8080`.
4. אפשר גישה למצלמה כאשר תתבקש.
5. הדפס את התבנית של Hiro marker (זמין באינטרנט) כדי לראות את המטבע מופיע עליה.

## הערות
- המשחק משתמש ב-AR.js וב-A-Frame להצגת מציאות רבודה.
- ניתן להוסיף עוד אובייקטים וירטואליים בקובץ `index.html`.
