# משחק מציאות רבודה (AR) - גרסת Iron Man Mk.III

משחק זה הוא סימולטור AR מתקדם המשתמש בטכנולוגיות דפדפן חדישות כדי לדמות את הממשק הטקטי של איירון מן.

## 🚀 תכונות מתקדמות (חדש ב-Mk.III!)

### 📊 מערכות מידע טקטיות (Real-Time Sensors)
המערכת מתחברת לחיישני הטלפון ומציגה נתונים אמיתיים בזמן אמת:
*   **מדד חשיפה (Hazard Meter):** אלגוריתם חכם המשלב נתוני קרינה אלקטרומגנטית (EMF) ועוצמת אור (Light) לחישוב רמת סכנה סביבתית (SAFE / CAUTION / DANGER).
*   **מצפן (Magnetometer):** תצוגת כיוון (Heading) מדויקת 360° עם סרגל ניווט נע.
*   **מד תאוצה (G-Force):** חישוב עומס כבידה בזמן אמת. מתריע על פגיעות (Impact) וזעזועים.
*   **GPS טקטי:** תצוגת גובה (Altitude) ומהירות תנועה (Speed) בקמ"ש.
*   **חיישן אור (Lux):** מדידת תאורה סביבתית והפעלה אוטומטית של **ראיית לילה** בתנאי חושך.
*   **מד טווח (Rangefinder):** ה-AI מחשב מרחק משוער לאובייקטים ורמת ודאות (Confidence) כשהכוונת ננעלת עליהם.

### 🧠 בינה מלאכותית (AI)
*   **זיהוי אובייקטים חכם:** המערכת מזהה אובייקטים (אנשים, טלפונים, כוסות) רק כאשר הכוונת (Reticle) מונחת עליהם פיזית.

### 🎙️ שליטה קולית וויזואליזציה
*   **אקולייזר "Majestic":** גרף שמע גדול וזוהר המגיב ברגישות גבוהה לדיבור.
*   **פקודות קוליות:**
    *   **"Fire" / "Shoot":** ירי בנשק (אם יש נעילה).

### 📱 חוויית משתמש (UX)
*   **ממשק פרימיום:** HUD נקי ללא מסגרות, עם ווידג'טים צפים.
*   **PWA (Full Immersion):** האתר פועל כאפליקציה במסך מלא, ללא פסים לבנים של הדפדפן (מומלץ להוסיף למסך הבית).

## 🕹️ הוראות הפעלה
1.  הפעל את השרת המקומי: `http-server ./ar_game -p 8080`.
2.  פתח בטלפון את הכתובת (חובה **HTTPS** או Localhost כדי לאפשר גישה לחיישנים).
3.  **לחץ על המסך** כדי לאתחל את המערכת ולאשר שימוש במיקרופון/מצלמה.
4.  הכנס את הטלפון לקסדת VR (או החזק ביד).
5.  הזז את הראש כדי לראות את המצפן והנתונים משתנים. דבר כדי לראות את האקולייזר.

## ⚠️ דרישות
*   דפדפן מודרני (Chrome לאנדרואיד).
*   הרשאות חיישנים פעילות (ב-iOS יש לאשר Motion Sensors בהגדרות).
*   חיבור אינטרנט לטעינת מודל ה-AI.
